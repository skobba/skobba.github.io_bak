# IOTstack

## Setup on rpi
```
sudo apt update
sudo apt upgrade -y
```

```
sudo apt install -y curl
```

```
sudo apt-get install screen
```

```
cd /home/pi/
screen
curl -fsSL https://raw.githubusercontent.com/SensorsIot/IOTstack/master/install.sh | bash
```


## pigpiod
Raspberry PI GPIO interface running on port 8888.
